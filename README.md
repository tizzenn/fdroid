# Repositorio F-Droid de tizzenn

Repositorio F-Droid personal. Las apps se compilan y firman automáticamente
con GitHub Actions y se publican aquí vía GitHub Pages.

## Cómo añadirlo a tu app de F-Droid

1. Abre F-Droid en el móvil → **Ajustes → Repositorios → +**
2. Añade esta URL (incluye la huella de verificación):

```
https://tizzenn.github.io/fdroid/repo?fingerprint=63D6489D8FC1E7D4076E52B127003666B7805D83189A4F83F4933BB72F8FB144
```

3. Busca **Lembra** o **Bote** e instálala. A partir de ahí, las actualizaciones
   llegarán solas como con cualquier otra app de F-Droid.

// Las apps

Lembra

Recordatorios de lo que caduca
  • Fichas con repetición flexible: cada X días, semanas, meses o años
  • Categorías con icono (coche, casa, niños, mascotas…) reordenables
  • Lista filtrable por categoría y ordenable por fecha (asc/desc)
  • Avisos con antelación configurable, resistentes a reinicios
  • Volcado opcional al calendario del sistema
  • Copia de seguridad exportable a JSON
  • Temas blanco/negro y colores a elegir
GPLv30 permisos raros~2 MB

Bote

Cuentas de grupo sin discusiones
  • Eventos con asistentes (de contactos o a mano) y foto

  • Sincronización con fusión inteligente: archivo, código QR o automática contra tu propia nube (Supabase gratuito o cualquier PostgREST; opcional, tú pones las llaves)
  • Detector de pagos opcional: la notificación del banco se convierte en apunte precargado
  • Modo extorsión opcional: «plata o plomo, me debes 12,50 €»

// Sincronización en la nube (opcional, Bote)

Bote funciona 100% en local; el QR y el archivo bastan para grupos que se ven la cara. Si además quieres que se sincronice solo, el servidor lo pones tú — nosotros no vemos nada, no hay cuenta central, no hay letra pequeña. Uno del grupo lo monta (5 minutos) y listo:

  1. Crea un proyecto gratuito en supabase.com (sin tarjeta).
  2. En SQL Editor, ejecuta esto y pulsa Run:
create table if not exists eventos_sync ( uuid text primary key, datos jsonb not null, actualizado timestamptz not null default now() ); alter table eventos_sync enable row level security; create policy "acceso anon" on eventos_sync for all using (true) with check (true);
  1. Copia la Project URL y la clave anon (Settings → API).
  2. Al crear el evento, en Sincronización elige Otro y pega la URL y la clave (o guárdalas en Ajustes como servidor Por defecto para tu grupo habitual).

Lo configura solo quien crea el evento. El servidor viaja dentro del evento al compartirlo (QR o archivo), así que los demás, al importarlo, quedan conectados solos — sin teclear nada. Cada grupo puede tener su propio servidor, así que puedes estar en varios grupos a la vez. A partir de ahí, al abrir un evento la app baja, fusiona y sube.

La clave anon es la llave del buzón del grupo: compártela por privado y no la publiques. ¿Un VPS propio con PostgREST en vez de Supabase? Misma tabla, cambia solo la URL.
